BackgroundTimer

public enum BackgroundTimer

Essentially a singleton you can never create an instance of because it has no cases. It provides a set of methods for scheduling blocks to run in the background.

  • schedule(interval:handler:)

    Schedules a block to run on the stadard queue for OysterKit.

    -Parameter interval: The delay before executing -Parameter handler: The block to execute -Returns: The resultant DispatchSourceTimer -SeeAlso: DispatchInterval.fromSeconds(seconds: Double)->DispatchTimeInterval for an easy way to create the interval

    Schedules a block to run on the named queue for OysterKit.
