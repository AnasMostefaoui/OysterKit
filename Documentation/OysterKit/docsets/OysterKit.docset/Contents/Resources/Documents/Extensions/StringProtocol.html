StringProtocol

protocol StringProtocol : BidirectionalCollection, TextOutputStream, TextOutputStreamable, LosslessStringConvertible, ExpressibleByStringLiteral, Hashable, Comparable where Self.Element == Character

A utility method to easily create a token stream from any string, allowing you to incrementally iterate over the results (lazily)

   ...
   for token in "PARSE ME".tokenStream(with: myRules){
       print("\(token)")
   }
