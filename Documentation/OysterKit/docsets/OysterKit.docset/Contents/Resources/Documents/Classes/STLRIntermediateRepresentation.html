STLRIntermediateRepresentation

public class STLRIntermediateRepresentation : CustomStringConvertible

This classes is a bespoke implementation of an IntermediateRepresentation used when parsing STLR. Unlike generic implementations it specifies a concrete data structure that is populated during parsing
